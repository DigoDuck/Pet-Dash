# PetDash

App web de gestão operacional e financeira do spa/estética animal **Ângelo Spa Animal**. Usuária única (Patricia), substitui o controle por planilha.

## Stack

- **Backend:** Django + Django REST Framework (`backend/`)
- **Frontend:** React (Vite) + Tailwind (`frontend/`)
- **Banco:** PostgreSQL
- **Deploy:** backend + Postgres no Railway, frontend no Vercel
